Step 0: 安全警告

OpenClaw 是一个个人 AI Agent，默认情况下是单用户信任边界。

这个 Bot 可以读取文件并在启用工具时执行操作。恶意提示可能诱使其做出不安全的行为。

默认情况下，OpenClaw 不是多租户边界。如果多个用户可以向一个启用了工具的 Agent 发送消息，他们共享该委托的工具权限。

如果不熟悉安全加固和访问控制，请不要运行 OpenClaw。

建议：

  • 使用配对/白名单 + 提及 gating
  • 多用户/共享收件箱：分离信任边界
  • 沙箱 + 最小权限工具
  • 定期运行: openclaw security audit --deep

必须阅读: https://docs.openclaw.ai/gateway/security
